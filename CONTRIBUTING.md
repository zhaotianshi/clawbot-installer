# 贡献指南

感谢你考虑为 OpenClaw Installer 做出贡献！

## 如何贡献

### 报告问题

如果你发现了 bug 或有功能建议：

1. 检查 [Issues](https://github.com/zhaotianshi/clawbot-installer/issues) 是否已有相关讨论
2. 如果没有，创建新的 Issue
3. 清晰描述问题或建议
4. 如果是 bug，请提供：
   - 你的环境信息（Android 版本、Termux 版本等）
   - 重现步骤
   - 错误信息或截图

### 提交代码

1. Fork 这个仓库
2. 创建你的特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交你的更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 开启一个 Pull Request

### 代码规范

- 使用清晰的变量和函数名
- 添加必要的注释
- 保持代码简洁易读
- 测试你的更改

### 文档

- 更新相关文档
- 保持中英文文档同步
- 添加必要的示例

## 开发环境

建议在以下环境中测试：

- Android 10+
- Termux (F-Droid 版本)
- Ubuntu (proot-distro)

## 问题？

有任何问题欢迎在 Issues 中讨论！
